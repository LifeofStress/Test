Digital Image

Steps to creating collage

First we open a new image in GIMP.
Next we will set the template to 600px * 400px.
Then we make the background color into the color like in the picture using eyedropper tools.
Now we will add the collage images by going to file and opening it as layers.
After we open one image as layer we will scale it to fit one corner of the template.
Then we repeat it to the other remaining two pictures.
Finally we will use add text to the bottom right corner of the template and scale it and modify the font.
